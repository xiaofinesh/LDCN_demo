## 第一部分 项目背景与现状分析

### 1.1 业务现状

贵方作为供电服务商，通过集装箱级蓄电池（5 MWh/块）替代传统柴油发电机，为中石油/中石化移动勘探钻井平台提供清洁电力。核心业务模式为：在充电站以电网电价充电，通过卡车运输至钻井平台供电，向油田客户按合同价售电，通过电价差实现盈利。

**当前运营关键参数：**

| 参数 | 数值 | 备注 |
|------|------|------|
| 单块电池容量 | 5,000 kWh (5 MWh) | 集装箱级电池舱 |
| 充电功率 | 1,725 kW | 充满约3小时 |
| 运输距离 | 30~60 公里 | 充电站↔钻井平台 |
| 换电耗时 | 约15分钟 | 热切换（先接后断） |
| 当前电池数量 | 3 块 | α-01 / β-02 / γ-03，轮转使用 |
| 充电站 | 1 家 | 河间市沙洼乡附近，10kV接入 |
| 钻井平台 | 1 个 | 任丘市于村乡附近（JH-017 井位） |

### 1.2 当前调度方式的三大风险

#### ⚠ 风险一：断电风险 — 高峰日3块电池余量极小

钻井平台要求不间断供电，但用电负荷波动极大。在高负荷日（日用电量超过25,000度），单块电池仅能供电约3.4小时，而电池离开平台到充满返回的周期约4.5~5.5小时。3块电池的轮转余量仅约0.4块，几乎无缓冲空间。一旦遭遇运输延迟或充电异常，将面临断电风险。

#### ⚠ 风险二：电费浪费 — 未充分利用峰谷电价差

河北电网实行五档分时电价，尖峰电价（1.07元/度）是深谷电价（0.37元/度）的 **2.92倍**。人工调度难以精确匹配充电时段与电价波谷，大量充电发生在平段甚至高峰时段，每度电多花 0.3~0.7 元。按日均充电11,302度计算，每天潜在浪费数千元。

#### ⚠ 风险三：规模化瓶颈 — 人工调度无法支撑扩展

当前1个平台、3块电池、1个充电站的规模尚可人工管理。但扩展至多平台后，调度变量呈指数级增长（每新增1个平台和3块电池，调度组合增加数千种），人工经验将无法保障调度质量。

| 指标 | 数值 |
|------|------|
| **日均用电量** | **11,302 度** |
| 最高日用电 | 33,810 度 |
| 波动系数（标准差/均值） | **80%** |
| 停工天数占比 | 12% |

### 2.2 负荷分布特征

| 发现 | 对调度的影响 |
|------|------|
| 88%的天数为中低负荷 | 3块电池绰绰有余，有充分的充电时段选择空间 |

| 存在连续高负荷期 | AI需识别趋势，提前调整充电策略 |

## 第三部分 电价优化空间测算

充电电费是运营成本的核心组成部分。河北电网实行五档分时电价，不同时段充电成本差异巨大。以下基于实际电价数据和用电数据，逐步推导AI调度可带来的成本节省。

### 3.1 五档分时电价分析

| 档位 | 电价 (元/度) | 相对深谷倍数 |
|------|:---:|:---:|
| **尖峰** | 1.0709 | **2.92×** |
| 高峰 | 0.9380 | 2.56× |
| 平段 | 0.6642 | 1.81× |
| 低谷 | 0.3904 | 1.06× |
| **深谷** | **0.3669** | 1.00× |

**关键价差：**
- 尖峰 vs 深谷：**2.92倍**，每度差价 0.70元
- 高峰 vs 深谷：2.56倍，每度差价 0.57元
- 平段 vs 深谷：1.81倍，每度差价 0.30元

适用电价：1-10千伏 单一制

### 3.2 四季峰谷时段分布

河北电网的峰谷时段因季节而异，直接影响充电时间窗口的选择。深谷和低谷时段集中在夜间和凌晨，是充电最优窗口。

### 3.3 成本节省测算

**测算方法：**

**第一步：确定无优化基准成本。** 假设无智能调度时，充电时段随机分布于各电价档位。根据河北电网各季节各时段的小时占比，计算加权平均电价约为 0.70 元/度。按日均充电量 11,302 度计算，基准日充电成本约 7,911 元。

**第二步：评估优化约束。** AI调度的目标是将充电集中于深谷/低谷时段，但受限于：
1. 电池充满需约3小时，深谷+低谷时段每日约7~10小时，时间窗口有限；
2. 电池轮转周期约束，部分充电时段不可自由选择；
3. 高峰日轮转紧张，可能被迫在非最优时段充电；
4. 运输和换电时间占用调度灵活性。

**第三步：保守估计优化比例。** 综合以上约束，保守估计AI调度可将充电加权平均电价从 0.70 元/度降至 0.50~0.55 元/度，对应充电成本降低约 **15%~25%**。

**节省金额测算（单平台）：**

| 场景 | 优化幅度 | 日节省 | 年节省 |
|------|:---:|:---:|:---:|
| 保守 | 15% | 1,187 元 | 35.6 万元 |
| **中性** | **20%** | **1,582 元** | **47.5 万元** |
| 乐观 | 25% | 1,978 元 | 59.3 万元 |

> 注：以上为单平台年化测算（按300天运营计）。扩展至多平台后，节省金额按平台数量线性增长。理论最大节省空间（100%尖峰 vs 100%深谷）为年239万元，上述保守估计仅为理论值的15%~25%，留有充分安全边际。

**✓ 核心结论：** 仅电价优化一项，单平台年节省即可达 35~59 万元。加上减少无效运输趟次节省的油费、降低断电风险避免的损失、以及运营效率提升释放的人力，系统的综合投资回报期预计在 1~2 年以内。

## 第四部分 解决方案概览

我们为贵方提供一套 **AI驱动的电池智能调度与运营管理平台**。系统实时采集电池状态和平台用电数据，通过AI算法自动生成最优调度方案，将指令下发给现场运维人员和卡车司机执行，同时提供全流程可视化监控。

### 4.1 系统整体流程

```
[数据采集层]          [AI调度引擎]              [指令下发层]
BMS电池数据    →    LSTM负荷预测      →    调度看板
GPS位置数据    →    MILP全局优化      →    司机App/小程序
电价数据        →    RL实时调整         →    告警通知
用电数据
```

### 4.2 AI调度引擎：三层算法协同

系统核心是AI调度引擎，采用"预测—规划—调整"三层算法协同工作：

| 算法层 | 技术 | 作用 | 类比 |
|--------|------|------|------|
| **用电预测** | LSTM/Transformer 深度学习模型 | 预测未来2~6小时用电负荷，为调度提供前瞻性输入 | 像天气预报，提前知道明天要多少电 |
| **全局规划** | MILP 混合整数线性规划 | 在已知约束下求解全局最优方案，决定何时充电、何时换电 | 像导航软件，规划最省钱的路线 |
| **实时调整** | DQN/PPO 强化学习 | 应对突发事件（功率突变、运输延迟），实时修正调度 | 像老司机，遇到堵车会临时改路 |

### 4.3 调度决策覆盖五大维度

| 决策维度 | 含义 | 优化目标 |
|----------|------|----------|
| 何时换电 | 根据在用电池SOC和用电预测，确定最佳换电时间 | 保障供电不断 |
| 换哪块电池 | 从备用电池中选择最优上平台电池 | 最大化供电时长 |
| 何时充电 | 结合峰谷电价和调度计划，确定充电启动时间 | 最小化充电成本 |
| 充多少电 | 根据下次使用时间和预期负荷，确定目标SOC | 平衡成本与安全 |
| 去哪充电 | 多充电站场景下选择最优充电站 | 综合距离与电价 |

### 4.4 面向规模化的架构设计

系统从设计之初即面向多平台、多充电站的大规模部署场景。一期交付的系统可无缝支持新增钻井平台和充电站，无需重新开发。当规模扩大到一定程度后，二期算法（多智能体协同调度 + 时空图网络）将进一步提升大规模场景下的调度效率。

| 维度 | 一期（当前） | 扩展后 |
|------|-------------|--------|
| 钻井平台 | 1个 | N个（按需扩展） |
| 电池数量 | 3块 | M块（共享电池池） |
| 充电站 | 1家 | K家（含自建站） |
| 调度算法 | 预测+MILP+RL | 升级为多智能体+时空图网络 |
| 优化维度 | 时间优化（何时充电） | 时间+空间联合优化 |

### 5.4 告警中心 | 四级告警管理 · AI 处置建议

告警中心采用四级分级体系：红色（紧急/断电风险）、橙色（预警/备用电池未就位）、黄色（提醒/充电进度滞后）、蓝色（信息/计划变更）。不同级别触发不同通知渠道：红色需即时电话+短信+App推送，橙色为短信+App，黄色仅App，蓝色为App内通知。红色告警会在详情页顶部显示专门的紧急响应横条，提供"呼叫现场"和"启动应急预案"按钮。每条告警均附带 AI 推荐处置方案（可一键执行）和完整的处置时间线（满足国央企审计要求）。

![告警中心界面](screenshots/alert_center.png)

**关键特性：**
- **四级分级告警** — 紧急/预警/提醒/信息 四档，通知渠道差异化配置
- **AI 推荐处置方案** — 每条告警附带 AI 生成的处置步骤，可一键执行
- **紧急响应横条** — 红色告警专属按钮：呼叫现场 + 启动应急预案
- **完整处置时间线** — 从告警产生到最终处置全流程留痕，支持审计追溯

### 5.5 电价优化分析报表 | AI 调度价值的量化呈现

这是财务总监最关注的页面。报表以超大数字 Hero 区域展示累计节省金额（含年化预测），下方通过 "成本对比"、"充电时段分布"、"每日节省趋势"、"年化收益预测" 四个维度，把抽象的 AI 调度成果翻译成直观的人民币。"优化空间分析" 卡片显示当前节省已达理论最大值的 38.7%，并给出提升建议（如增加第 4 块备用电池）。所有数据基于 BMS 实时数据和真实电价，可随时导出 PDF。

![电价优化分析报表界面](screenshots/pricing_report.png)

**关键特性：**
- **Hero 节省金额** — 当月累计节省 + 年化预测 + 节省比例一屏展示
- **四维价值拆解** — 成本对比 / 时段分布 / 每日趋势 / 年化预测
- **场景化预测** — 保守/中性/实际/乐观 四档年化收益对照表
- **一键导出 PDF** — 财务可随时导出月度报表，用于内部汇报与决算
